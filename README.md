# Community Assistance & Volunteer Matching System - Frontend

## Overview
A comprehensive MERN stack platform connecting community members with verified volunteers across multiple service categories including elder care, child care, household chores, repairs, tutoring, and cooking help.

## Features
- User and Volunteer authentication with Indian mobile number validation
- Category-based volunteer search and filtering
- Booking system with confirmation modal
- Real-time arrival notifications
- Dedicated dashboards for both users and volunteers
- Volunteer profiles with experience, ratings, and certifications
- Responsive and modern UI

## Tech Stack
- React 18
- React Router DOM v6
- Axios for API calls
- React Icons
- React Hot Toast for notifications
- React Modal
